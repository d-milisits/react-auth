# About react-auth

**Version 1.0.0**

React.js that utilizes Firebase authentication and allows users to create an account/login.

## How to run locally

- Simply clone the project, CD into the directory
- Run 'npm install' in your terminal
- 'npm start'
- Begin authenticating!
